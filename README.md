# HR Employee Attrition Analytics

## 📌 Project Overview

This project analyzes employee attrition using SQL, Python, and Power BI.

The objective is to identify factors affecting employee turnover and provide data-driven business insights.

## 🛠️ Tools Used

- MySQL
- Python
- Pandas
- Matplotlib
- Power BI
- GitHub
